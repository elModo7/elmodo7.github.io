Notification System Demo (Godot Engine)

Note: In order to hear sound you have click or interact in some way with the iframe.

GitHub Repo Godot Notification System Demo

This project can act as a client or server depending on requirements. It works either via raw TCP sockets or websockets so it can also be deployed for the web, like in this example.
It basically serves as a way of showing animated notifications on streaming platforms, with particle effects, sound, glowing and using custom animations for juicing up the stream.
You just have to send a JSON object and the alert will be displayed.

{
    "imageScale": "Vector2(0.25, 0.25)",
    "img": "C:\\Users\\victo\\Desktop\\800px-Clefairy.png",
    "message": "Who's that Pok\u00e9mon?!",
    "particlesModulation": "Color(1, 0, 0, 0)",
    "soundPath": "C:\\Users\\victo\\Documents\\Tweener Remote Alerts Notification System\\sound\\who-is-that-pokemon-its-pikachu-vine.mp3",
    "type": "whoIsThatPokemon",
    "volumeDB": "-10"
}
I have also implemented a queue system so that a stream of alerts can be received and displayed sequentially.
I also use it personally on a mini screen that I have attached to a micro server for custom notifications.
